# OMDB API Wrapper

OMDB API Wrapper let developers interact with OMDB API.
Deployed on Heroku:  
##

```
Server URL: https://radiant-taiga-77307.herokuapp.com/
```

This API consists of two endpoints

- GET /api/v1/movies/[id]
- GET /api/v1/search?{title,year,type,page}

#
- Ruby version
   ruby 2.6.9p207
   
- Rails version
   Rails 6.0.4.4

## Installation


```bash
bundle install
```
## Test

```bash
rake test
```
## Server

```bash
rails s
```
